# 跑步装备选择指南

## 跑鞋
跑鞋是唯一不可或缺的装备。新手选鞋要点：
- 比日常鞋大半码（跑步时脚会肿胀）
- 鞋头要有足够空间让脚趾活动
- 根据足型选择：扁平足选支撑型，高足弓选缓震型，正常足选中性鞋
- 建议去专业跑鞋店测试 gait analysis
- 跑鞋寿命约 500-800 公里，到期后即使外观完好也要更换

## 服装
- 上衣：速干面料（聚酯纤维），避免纯棉（吸汗后变重、摩擦皮肤）
- 裤子：跑步短裤或紧身裤，避免宽松长裤
- 袜子：跑步专用袜，避免纯棉袜（容易起水泡）
- 女性：高强度运动内衣

## 配件
- 运动手表/手环：记录距离、配速、心率（非必须但有用）
- 臂包/腰包：放手机和钥匙
- 水壶：超过 30 分钟建议补水
- 反光装备：夜间跑步必备
