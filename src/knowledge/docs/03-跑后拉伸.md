# 跑后拉伸指南

跑后拉伸应在跑步结束后 5 分钟内开始，此时肌肉温度最高，拉伸效果最好。

## 标准拉伸流程（10 分钟）

每个动作保持 15-30 秒，每侧做 2 组。不要弹震，感到轻微拉伸感即可，不要到疼痛。

### 大腿前侧（股四头肌）
站立，一手扶墙，另一手抓住同侧脚踝，将脚跟拉向臀部。保持膝盖并拢。

### 大腿后侧（腘绳肌）
坐在地上，一腿伸直，另一腿弯曲脚底贴在大腿内侧。身体前倾，双手伸向伸直腿的脚尖。

### 小腿（腓肠肌和比目鱼肌）
面对墙壁，双手推墙，一腿前弓一腿后伸，后腿伸直脚跟不离地。然后后腿微曲，拉伸更深层肌肉。

### 髋屈肌
单膝跪地，前腿弓步，骨盆前推，感受后腿大腿前侧根部的拉伸。

### 臀部（梨状肌）
仰卧，一腿脚踝搭在另一腿膝盖上方，双手抱住下方大腿拉向胸部。

### 下背部
跪姿，臀部坐向脚跟，双手向前伸展，额头贴地（婴儿式）。
